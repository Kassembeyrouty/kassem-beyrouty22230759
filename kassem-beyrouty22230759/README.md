# Art Gallery Website

This is my second phase project for CSCI390 Web Programming. I took my old HTML and CSS art gallery website and rebuilt it using React. The idea stayed the same but I added more pages, better design, and some useful features like a ticket price calculator.

## What the project does

The website is an art gallery where you can browse different artworks, filter them by category, calculate how much tickets would cost for an event, and contact the gallery. It has 4 pages in total.

## Pages

- Home – introduction to the gallery with a hero image
- Gallery – shows the artworks with a filter by category button
- Services – describes what the gallery offers plus a ticket calculator
- Contact – a contact form with the gallery location and info
